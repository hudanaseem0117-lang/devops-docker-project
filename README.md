## 🔧 Docker Compose
docker compose up --build

## 📊 Monitoring & Troubleshooting
docker ps

## ⚙️ How it Works
This project uses Nginx inside a Docker container to serve a custom HTML page. 
Docker Compose is used to manage the container easily.
